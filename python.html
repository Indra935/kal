5.1 Komputasi Sistem Persamaan Linier

Penyelesaian Sistem Persamaan Linier dalam skala besar memerlukan bantuan komputasi. Menggunakan bahasa pemrograman Python dan library NumPy, kita dapat melakukan Operasi Baris Elementer (OBE) secara sistematis dan cepat.

5.2 Definisi Fungsi Operasi Baris (Python)

Berdasarkan kaidah aljabar, kita mendefinisikan fungsi dasar untuk memanipulasi baris matriks teraugmentasi:


import numpy as np

# Fungsi Penukaran Baris (Row Swap)
def RowSwap(mat, r1, r2):
    mat[[r1, r2]] = mat[[r2, r1]]
    return mat

# Fungsi Perkalian Skalar (Row Scale)
def RowScale(mat, row, k):
    mat[row] = mat[row] * k
    return mat

# Fungsi Penjumlahan Kelipatan (Row Addition)
def RowAdd(mat, source, target, k):
    mat[target] = mat[target] + k * mat[source]
    return mat

5.3 Implementasi Kasus dan Output

Berikut adalah penerapan fungsi tersebut pada sistem persamaan yang ada di materi (Slide 84):


# Matriks Augmented awal
A = np.array([
    [1., -1., 1., 3.],
    [2., 1., 8., 18.],
    [4., 2., -3., -2.]
])

# Eksekusi OBE
A = RowAdd(A, 0, 1, -2) 
A = RowScale(A, 1, 1/3)
A = RowAdd(A, 1, 2, -6)

print("Matriks Eselon Baris Akhir:")
print(A)
